Core IT - Intermediate

Course Title:

Core IT - Intermediate level for students aged 11+.

Course Overview:

The Core IT - Intermediate level course covers a range of advanced topics and subjects to build upon the foundational knowledge gained in the Entry level. Here is a lesson plan with some of the topics and subjects typically covered in this course:

  1. Networking Fundamentals:
    • Review of basic networking concepts
    • Introduction to advanced networking protocols
    • Subnetting and IP addressing
    • Hands-on network configuration exercises
  2. Operating Systems:
    • In-depth study of operating system functions
    • Introduction to Linux and Windows server administration
    • File system management and permissions
    • Virtualization technologies
  3. Web Development:
    • HTML5 and CSS3 for web design
    • Introduction to JavaScript and front-end frameworks
    • Server-side scripting languages (e.g., React)
    • Building dynamic web applications
  4. Cloud Computing:
    • Cloud service models (IaaS, PaaS, SaaS)
    • Introduction to popular cloud platforms (e.g., AWS, Azure, Google Cloud)
    • Cloud deployment and management
    • Scalability and cost considerations
  5. Introduction to Python:
    • Basics of Python programming language
    • Data types, variables, and control structures
    • Writing Python scripts for automation
    • Introduction to libraries and frameworks
  6. Emerging Technologies:
    • Introduction to AI, machine learning, and data science
    • IoT (Internet of Things) and its applications
    • Blockchain technology overview
    • Discussion of the latest trends in IT

Course Format and Structure:

The course is conducted entirely online through the Slack platform. The lessons will be hands-on and highly engaging and interactive. One group will have maximum 5 students . Each lesson has a duration of approximately 50 minutes , which is roughly equivalent to one hour, twice per week. The Core IT - Intermediate course typically spans a period of 6 months, although the exact duration may vary from one student to another. The primary difference between the Intermediate and Entry levels of Core IT lies in the fact that I will skip the fundamental aspects of general IT and dive straight into advanced technologies that are currently in use. I will illustrate how the world of IT is constructed and emphasize the essential knowledge required. Each student will receive personalized attention and support throughout the course.

Course Materials and Resources:

Given that the lessons will be held online, as previously mentioned, course materials will be accessible through screen sharing. These materials consist of PowerPoint presentations that are regularly updated with the latest information. The key essentials for students will include owning a laptop and having a notebook at their disposal. The software they are going to use during lessons will be absolutely free to use.

Assessment and Grading:

Quizzes: (every 2 weeks) Regular quizzes will be administered to assess students' comprehension of course material and to reinforce key concepts. These quizzes will typically cover recent topics and serve as a form of ongoing assessment.
